Tagbox

This is tagbox, a tagging plugin for jQuery.

In this example, you can type some colour names to see what happens. (see code)

$('.tag_colours').tag_box({ keyup: function(e){ clearTimeout(this.colorTimeout); this.colorTimeout = setTimeout(function() { e.target.parentNode.className = e.target.value.replace(/[-]/gim," ").toLowerCase(); }, 300); } });
